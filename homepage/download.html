Download

The current stable version is 2.7.4. You can grab it from berlios.de right here

The current development version is 2.8.0rc1. You can grab it from berlios.de as well.

You can also obtain the most recent development code from our SVN repository. Here is what you need to do:

Fetch the stable SVN trunk:
The repository can be checked out through anonymous (svnserve) SVN.

$ svn checkout svn://svn.berlios.de/noteedit/

Only project developers can access the SVN tree via SSH. SSH2 must be installed on your client machine. Enter your site password when prompted.

$ export SVN_SSH="ssh -l developername"

$ svn checkout svn+ssh://svn.berlios.de/svnroot/repos/noteedit/

Fetch the unstable SVN trunk:

$ svn checkout svn://svn.berlios.de/noteedit/unstable

$ svn checkout svn+ssh://svn.berlios.de/svnroot/repos/noteedit/unstable
