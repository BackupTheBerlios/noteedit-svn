The current stable version is 2.8.0. You can grab it from berlios.de right here

There is no current development version. We have set up everything for future 2.9 unstable versions. You can grab them from berlios.de as soon as they are available.

For older releases and other files, please visit our files section at the Berlios project page.

You can also obtain the most recent development code from our SVN repository. Here is what you need to do:

Fetch the latest stable SVN version:

The repository can be checked-out through anonymous (svnserve) SVN:

$ svn checkout svn://svn.berlios.de/noteedit/branches/stable-2.8 noteedit

Project developers can access the SVN tree via SSH as well. SSH2 must be installed on the client machine! Enter your site password when prompted.

$ svn checkout svn+ssh://DEVELOPER_NAME@svn.berlios.de/svnroot/repos/noteedit/\
  branches/stable-2.8 noteedit

Fetch the latest development SVN version:

Again, both anonymous and SSH access to SVN are available. 
$ svn checkout svn://svn.berlios.de/noteedit/trunk noteedit
$ svn checkout svn+ssh://DEVELOPER_NAME@svn.berlios.de/svnroot/repos/noteedit/\
  trunk noteedit
