About NoteEdit

NoteEdit is a music score editor that supports an unlimited number of staffs, an unlimited length of staffs, up to 9 voices per staff, a MIDI playback of written notes, a number of import and export filters to many formats like MIDI, MusicXML, ABC Music, PMX, MusiXTeX and LilyPond and more!



For more detailed information about the project, please visit our project site at Berlios!
